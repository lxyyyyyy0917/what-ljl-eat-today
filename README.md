# 🍱 李建霖今天吃什么？

一个可爱的午餐选择助手，帮助选择困难症决定今天吃什么！

## ✨ 特性

- 🎲 **随机选择** - 让命运帮你决定今天吃什么
- ➕ **自定义餐品** - 添加你喜欢的所有餐品
- 🎨 **精美设计** - 现代化的渐变色卡片设计
- 📱 **PWA 支持** - 可以添加到手机主屏幕，像原生 App 一样使用
- 💾 **本地存储** - 数据保存在本地，不会丢失
- 🌐 **离线可用** - 即使没有网络也能使用
- 🚀 **快速添加** - 12 个常见餐品一键添加

## 🎯 在线访问

访问地址：[https://lxyyyyyy0917.github.io/what-eurekaliao-eat-today/](https://lxyyyyyy0917.github.io/what-eurekaliao-eat-today/)

## 📱 如何使用

1. 点击"添加"按钮或使用快速添加按钮添加餐品
2. 点击"开始选择"按钮
3. 系统会随机为你选择一个餐品
4. 不满意？点击"再选一次"重新选择

## 🛠️ 技术栈

- 纯静态 HTML/CSS/JavaScript
- Tailwind CSS - 样式框架
- PWA - 渐进式 Web 应用
- Service Worker - 离线支持
- LocalStorage - 本地数据存储
